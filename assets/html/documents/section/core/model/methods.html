Method

Channel 註冊在 JSB.core.model ， 呼叫如 JSB.core.model.getObjKey()

Name
getObjKey(obj,key)

對目標Object資料取得資料

Parameters

  • obj : (Object)

    目標Object

  • key : (String)

    搜尋的key

Example 
										var obj = {
											a:{
												b:{
													name:"helloworld!"
												}
											}
										}

										alert(JSB.core.model.getObjKey(obj,"a.b.name"));

Returns
Misc

getObjtDiff(obj_org,obj_mod,type)

比對兩個Object資料的不同之處

Parameters

  • obj_org : (Object)

    原始資料

  • obj_mod : (Object)

    修改資料

  • type : (String) 可以設定 add ，del，modify，all

    比對的類型

Example 
										var obj_org = {
											a:{
												b:{
													name:"hello!"
												}
											}
										}

										var obj_mod = {
											a:{
												b:{
													name:"helloworld!"
												}
											}
										}

										console.log(JSB.core.model.getObjtDiff(obj_org,obj_mod,"modify"));

Returns
Misc
